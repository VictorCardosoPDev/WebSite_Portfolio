# Portfólio

## Visão Geral

Este repositório contém o código-fonte do meu portfólio pessoal. Desenvolvido com **HTML, CSS e JavaScript puros**, ele serve como um cartão de visitas digital para apresentar minha trajetória profissional, habilidades técnicas e projetos desenvolvidos.

**O objetivo principal deste projeto é:**
* Exibir minha **formação acadêmica** e **experiências profissionais**.
* Destacar meus **principais projetos** de desenvolvimento.
* Servir como um ponto de contato direto.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as tecnologias fundamentais da web:

* **HTML5:** Para a estruturação semântica do conteúdo.
* **CSS3:** Para o *design* responsivo e estilização visual.
* **JavaScript (ES6+):** Para interações dinâmicas, animações e manipulação do DOM.

## 💻 Como Visualizar Localmente

Como este é um projeto puramente estático (não requer servidor *backend* ou instalação de dependências), você pode visualizá-lo facilmente em qualquer navegador:

* **Acesse:** https://victorcardosopdev.github.io/Portfolio_WebSite/

## 📝 Estrutura do Conteúdo

O portfólio é dividido nas seguintes seções:

* **Home:** Introdução e boas-vindas.
* * **Projetos:** Galeria de projetos com links para seus respectivos códigos e demonstrações.
* **Formação** Detalhes sobre a formação e *hard/soft skills*.
* **Experiência:** Linha do tempo das experiências profissionais.
* **Contato:** Formulário ou links diretos para e-mail e redes sociais (LinkedIn, etc.).
